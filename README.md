# Career Guidance Chatbot
A simple interactive chatbot that helps users with career guidance.

## Features
- Career path suggestions
- Skill recommendations
- Resume guidance
- Interview preparation tips
- Salary insights

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Purpose
This chatbot helps students and job seekers explore career options and improve their preparation.

## Future Improvements
- Database integration
- AI/NLP integration
- User authentication
- Better UI design
